# Art_Shop_ground_vue

최종 완료 후 이미지

## 프로젝트 목표
- Vue에 쓰여지는 다양한 메서드와 더미 api를 활용하여 데이터 바인딩을 하고 vue에 대한 전반적인 지식을 습득하기 위한 연습용 프로젝트 입니다.

<br />

## 프로젝트 내용
- 미술 작품을 쇼핑하고 본인이 좋아하는 예술 작품을 다이어리에 공유하는 간단한 웹 사이트 입니다. 

<br />

## 사용 스택
- Vue, axios, bootstrap, vue-router, css, vuex

<br />

## 기능 구현 및 시연 영상
-  렌딩 페이지

-  shop 페이지
  -  정렬 기능
  -  api 호출(더보기)
  -  상세페이지 (유효성 검사)
-  다이어리 페이지
  -  이미징 업로드 및 글 발행
  -  좋아요 기능
